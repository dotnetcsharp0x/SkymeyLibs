# SkymeyLibs

appsettings.json
```csharp
{
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "AllowedHosts": "*",
  "MainSettings": {
    "MongoDatabase": {
      "DBName": "dbname",
      "DBServer": "mongodb://127.0.0.1:27017"
    }
    }
}
/*
"DBName": "dbname" = database name
"DBServer": "mongodb://127.0.0.1:27017" = server ip
*/
```
